# GridAlpha Design Reference

These files are exported from Figma Make and serve as the 
visual specification for GridAlpha Terminal V2.

## v1 — Card-Heavy System
Component library built around card containers. Reference 
for: chart treatments, table layouts, controls, map 
components, overlay patterns.

## v2 — Chromeless System  
Component library built around three composition modes: 
HERO (no chrome), FLOW (rule + eyebrow only), CONTAINED 
(cards reserved for DominantCard and CompactCard only). 
Reference for: number hierarchy, label/badge system, 
Peregrine feed components, two-state navigation, 
composition philosophy.

## Usage
When implementing any terminal component, read both files 
first. v2 takes precedence on composition decisions. v1 
takes precedence on chart density and map components.

Do not import these files. Do not compile these files.
They are read-only design references.
